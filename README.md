# StoatBridge

StoatBridge is a web-based toolkit that helps Discord community owners migrate their servers to the Stoat platform. It provides a guided transfer flow, channel/role mapping, and a visual Embed Studio for crafting rich Stoat embeds — all from one dashboard.

**Live site:** [stoatbridge.com](https://stoatbridge.com)

## Features

- **Server Transfer** — Step-by-step guided flow to scan, map, and transfer a Discord server to Stoat
- **Embed Studio** — Visual editor to build, preview, and send Stoat embeds with live JSON output
- **Channel & Role Mapping** — Map channels and roles between source and target servers
- **Bot Advisor** — Get recommendations for bots that fit your community
